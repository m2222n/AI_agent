2주차 과제 – Tech-GPT 프로토타입

이 폴더에는 2주차 과제를 위한 AI 에이전트 프로토타입 코드와 보고서가 포함되어 있습니다.

구성:
- app.py : Streamlit 기반 메인 애플리케이션 (Tech-GPT 데모)
- report_2week_taemin.md : 2주차 과제 보고서 (마크다운)
- requirements.txt : 필요한 Python 패키지 목록

실행 방법:
1) 가상환경 생성 및 활성화 (선택)
2) pip install -r requirements.txt
3) 환경변수 OPENAI_API_KEY 설정
4) streamlit run app.py
